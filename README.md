# Task 5: Exploratory Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Sample Superstore
dataset. The analysis uses statistical summaries and visualizations to identify
sales patterns, profitability trends, relationships, and anomalies.

## Dataset

The dataset contains 9,994 records and 21 columns covering:

- Orders and shipping details
- Customer segments and locations
- Product categories and sub-categories
- Sales, quantity, discount, and profit

No missing values or duplicate rows were found.

## Tools and Libraries

- Google Colab
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed

- Dataset structure and descriptive statistics
- Missing-value and duplicate checks
- Univariate analysis using histograms and boxplots
- Categorical feature analysis using value counts
- Correlation analysis using a heatmap
- Sales and profit relationship analysis
- Quantity and sales relationship analysis
- Discount and profit relationship analysis
- Top states by sales
- Top sub-categories by sales
- Monthly sales and profit trends
- Profit by sub-category
- Multivariate analysis using pairplots

## Key Findings

- Technology generated the highest total sales and profit.
- California was the top-performing state by sales.
- Phones and Chairs generated the highest sub-category sales.
- Copiers generated the highest sub-category profit.
- Tables produced the largest overall loss despite having high sales.
- Sales and Profit had a moderate positive correlation of approximately 0.48.
- Higher discounts were generally associated with lower profitability.
- Sales and Profit contained several notable outliers.

## Recommendations

- Continue focusing on high-performing Technology products.
- Review pricing and discount policies for loss-making products.
- Investigate the high sales but negative profit generated by Tables.
- Strengthen marketing in profitable states while reviewing costs in
  high-sales, low-profit regions.
- Monitor monthly sales and profit trends when planning promotions and
  inventory.

## Conclusion

The analysis highlights important differences between sales and profitability.
High sales do not always produce high profit, especially when discounts or
unprofitable product groups are involved. These findings can support better
pricing, discount, marketing, and product-management decisions.
